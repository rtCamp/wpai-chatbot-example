# WPAI Chatbot - WordPress Plugin

## System Requirements

- **PHP** 8.1+
- **WordPress** 6.7+

## Installation

See [DEVELOPMENT.md](DEVELOPMENT.md#setup-plugin-locally) for instructions on how to install and run the plugin locally.

## Quick Start

1. Go to your WP Admin Dashboard > Plugins and activate the plugin.
2. Go to the settings page, and update the iframe URL to point to `localhost:<chat-app-port>/chat/{chat-id}` (e.g. `http://localhost:3000/chat/rtcamp`).
3. Save the settings.
