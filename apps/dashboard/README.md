# WPAI_Chatbot - Dashboard Application

This is a dashboard application built using [Next.js](https://nextjs.org). It provides an interface for managing chats, users, and other administrative tasks.

## Quick Start

```bash
# Install dependencies
npm install

# Start the development server
npm run dev
```
